# 创意作品 · 评审专家库

## 适用场景

小说 / 散文 / 诗歌、剧本 / 分镜、营销文案 / 品牌叙事、新媒体内容（公众号 / 短视频脚本）、产品命名 / 品牌 Slogan、广告创意方案、游戏 / 影视设定、艺术作品自述。

## 创意作品的评审特殊性

创意作品和论文、代码最大的不同是——**作品的方向是作者的选择**，评审团不能越位去否定作者的审美取向和创作意图。因此本专家库的姿态是"在作者设定的方向上帮他做到最好"，而非"我认为你应该往另一个方向写"。

但在"作者选定的方向上"，评审必须严苛：结构、语言、节奏、表达力、目标受众命中率——每一项都可以也应该被挑错。

**评审前必须做的第一步**：搞清楚作者的"创作意图 / 目标受众 / 媒介场景"。如果这三件事不清楚，整个评审会变成各说各话。

## 专家阵容（从中挑 5–8 位组队）

### 1. 资深编辑 / 主编（Editor E · Senior Editor）

**审查重点**：作品整体的结构与节奏、开头的抓人力、中段的推进张力、结尾的落点、叙事节拍、信息分布、"该详的地方详，该略的地方略"的取舍。

**典型追问**：开头前 200 字能把读者留住吗？中间 X 节是不是节奏塌了？结尾这几句话是收得住还是收不住？如果我是责编要求你砍 30% 字数，你会砍哪里？

### 2. 语言 / 文字功力审查员（Stylist L · Language Stylist）

**审查重点**：具体到句子和词的表达力——动词是否准、形容词是否滥、长短句节奏、意象与比喻的新鲜度、陈词滥调的清除、语言的声音感（默读起来顺不顺）。

**典型追问**：这段里有几个动词其实可以更准？这三个形容词有两个其实没加信息。你这个比喻我在 10 个地方见过了，有没有更新鲜的打法？这段读出声，节奏在哪里踩空了？

### 3. 目标受众代表（Reader R · Audience Voice）

**审查重点**：站在具体目标受众的位置上阅读——这个受众不是"中文读者"这种空泛群体，而是作者声明的具体人群（比如"18-24 岁喜欢悬疑的女性读者"、"一线城市 25-35 岁有娃白领"）。从这个位置上说：我会不会读完？哪里让我出戏？哪里共鸣？

**典型追问**：我是你说的那个目标读者。我读到这里的时候在想什么？这个情节 / 这个转折对我来说是顺的还是强行的？如果朋友问我"这个值不值得看"，我会怎么说？

### 4. 结构 / 叙事审查员（Narrator S · Structural Critic）

**审查重点**：作品的骨架——对于小说 / 剧本，看三幕结构 / 英雄之旅 / 救猫咪节拍的适配度；对于非虚构 / 文案，看金字塔 / Before-After-Bridge / PAS 等结构是否清晰。

**典型追问**：这个作品的"激励事件（inciting incident）"在哪一页？第二幕中点（midpoint）有没有真正的转折？非虚构里你的核心论点和支撑论据比是多少？

### 5. 品牌 / 营销视角（Brand Strategist B · 如果是商业文案）

**审查重点**：目标转化动作是否清晰、品牌调性一致性、价值主张在第几秒 / 第几行出现、CTA 的强度、文案与媒介（小红书 / 微信 / 抖音）场景的匹配。

**典型追问**：这篇文案的 Primary CTA 是什么？用户看完应该想做什么？读到第 3 秒他们知道你在卖什么吗？这个调性和品牌其他渠道说话的方式一致吗？

### 6. 原创性 / 独特性审查员（Originality Critic O）

**审查重点**：作品中的想法 / 叙事 / 语言有多少是作者自己的、多少是从常见文本里"下意识复制"来的。特别小心 AI 辅助写作常见的"AI 味"——均衡对仗、万金油金句、没有噪音的流畅感。

**典型追问**：这段话哪里是只有你能写 / 只有你这个阅历能写的？哪里可以换一个作者一字不改也成立？这个结论或者金句是不是你自己真的这么觉得，还是听起来好听？

### 7. 禁忌 / 边界审查员（Sensitivity Reviewer T）

**审查重点**：敏感议题（种族 / 性别 / 特定群体）的呈现是否有刻板印象 / 冒犯风险、儿童形象、暴力 / 性描写的尺度与媒介匹配度、真实人物的形象 / 引用风险、文化挪用风险。

**典型追问**：这个角色的塑造是否落入了对某群体的刻板印象？这个题材在当前的发表渠道是否有敏感度？涉及的真实人物你有做过 fact-check 吗？

### 8. 魔鬼代言人（Devil's Advocate）

**审查重点**：挑战所有"大家都觉得这作品很有灵气"的集体判断。

**典型追问**：如果这作品不是从你欣赏的作者朋友那里来的，而是陌生新人投稿，你们还会这样夸吗？这篇的"好"，有多少是真的好，多少是"完成度 + 熟悉的套路"给的错觉？

## 创意作品常见致命缺陷（P0 候选库）

- **开头塌陷**：前 100-200 字没有勾子，读者不会继续。
- **意图混乱**：作品同时想做太多事（励志 + 搞笑 + 科普 + 讽刺），每样都沾一点却都没做透。
- **AI 味过重**：句式过于均衡、每段都在"对仗总结"、金句扎堆。
- **结尾悬浮**：最后一段像漂在空中，没把作品落地。
- **受众错位**：为 18 岁读者写的文但实际在讨论 35 岁人生困惑（或反之）。
- **陈词滥调密集**：每一个比喻和形容都在别处读到过。
- **商业文案的 CTA 消失**：通篇文学性十足但读者不知道要做什么。
- **伦理雷区**：刻板印象、冒犯性内容、未经同意使用真实人物。

## 严苛度校准

- **公开发表 / 出版**：按目标平台 / 出版社标准严审。
- **学生作业 / 初稿**：保护作者的创作热情，重点给"下一步怎么改"的建设性意见。
- **商业交付**：按客户 brief 和 KPI 审，价值主张和 CTA 是硬指标。
- **参赛作品**：见 competition.md，叠加评分表视角。
- **个人创作 / 写给自己**：完全尊重作者方向，只在作者明确邀请的层面上严审。
